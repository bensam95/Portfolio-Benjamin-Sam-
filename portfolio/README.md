# Portfolio - BTS SIO SLAM

Petit portfolio statique pour présenter des projets, expériences, une veille technologique et l'épreuve de portfolio.

Structure:
- `index.html` - page d'accueil
- `projects.html` - page des projets (avec affichage d'extraits de code)
- `experience.html` - stages et expériences
- `veille.html` - présentation de la veille technologique et outils
- `epreuve.html` - page dédiée à l'épreuve avec emplacement pour PDF
- `contact.html` - page contact
- `css/style.css` - styles
- `js/scripts.js` - interactions minimales

Usage local:
1. Ouvrez `index.html` dans votre navigateur (double-clic ou glisser-déposer dans le navigateur).
2. Pour intégrer votre PDF d'épreuve, placez le fichier dans `assets/tableau.pdf` ou modifiez le chemin dans `epreuve.html`.

Personnalisation rapide:
- Remplacer textes, images et coordonnées dans les fichiers HTML.
- Ajouter vos projets réels et remplacer les extraits de code.
- Pour publication, hébergez sur GitHub Pages ou un serveur web simple.

Bonne personnalisation !
