Gestionnaire de tâches (PHP / MySQL)

Application web pour gérer des tâches avec connexion, listes et priorités.

Démonstration 
<?php
// connexion simple PDO
$pdo = new PDO('mysql:host=localhost;dbname=todo','user','pass');
$stmt = $pdo->prepare('SELECT * FROM tasks WHERE user_id = ?');
$stmt->execute([$_SESSION['user_id']]);
$tasks = $stmt->fetchAll(PDO::FETCH_ASSOC);
?>
